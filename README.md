# Vue-Mine 扫雷

本项目最初是于学习jQuery时产出的一个小游戏，现以vuejs重构


试试：
[demo](https://enpitsulin.github.io/Vue-minesweeper/dist/)


## 目标
+ ~~以三大主流框架重构~~（Vuejs）
  + ~~基本数据结构~~
  + ~~基本游戏框架~~
  + 更好的UI
  + 更好的游戏性
    + 难度选择
      + 三级默认难度
      + 自定义难度
    + 游戏结束提示(遮罩层或toast消息)
  + 更好的代码逻辑
  + 更好的样式
  + .....
  
## 运行

### 项目设置

克隆或下载本仓库到本地 然后运行
```shell
npm install
```
### 启动服务器
```shell
npm run serve
```
然后浏览器打开
```
http://localhost:8080/
```
